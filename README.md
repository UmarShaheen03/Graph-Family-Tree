# Graph-Family-Tree
An interactive graph database visualization web application for efficient long lasting family tree management using Neo4j.

## Setting up the project development environment

-   Create the development environment using `python3 -m venv .venv`. 
-   Activate the virtual environment folder using `source .venv/bin/activate`. 
-   Install project dependencies using `pip install -r requirements.txt`.

## Running the project

After setting up the virtual environment and installing the dependencies, run `flask --app main run`
